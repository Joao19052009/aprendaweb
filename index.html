TUTORIAL INFORMAÇÕES FALE CONOSCO

ENSINO BASICO DE WEB

O trabalho apresentou uma abordagem prática e interativa para introduzir as linguagens HTML, CSS e Java, mostrando que ferramentas simples e exemplos visuais são eficazes para engajar

iniciantes no aprendizado de programação. Os resultados destacaram que a criação

de projetos básicos, como páginas web personalizadas, não apenas facilita a

compreensão dos conceitos fundamentais, mas também estimula a curiosidade

dos participantes em explorar mais profundamente o campo da tecnologia.
